# Guess the character: Rick and Morty
Um joguinho simples feito somente com HTML, CSS e Javascript que utiliza a Rick and Morty API para selecionar um personagem aleatório e desafiar o jogador a acertar o nome deste personagem (Está em inglês).

Uma requisição busca todos os personagens da API para construir o datalist do input, a partir disso uma requisição é feita para selecionar um único personagem que terá a foto exibida e será o personagem que o usuário deve acertar.

Feito com o propósito de brincar um pouco com API e lógica de jogo.
